This project is about creating a webpage with html, java, javascript and css. This site has some Java functionality combined with ArcGIS to display a map on the weather tab. On this page, it is possible to search for a species and provide certain options, see the weather page. The program also searches for environmental variables of the species you're looking for. With this data, it searches for places on the earth that have similar conditions and where this species also could possibly live (According to environment)
